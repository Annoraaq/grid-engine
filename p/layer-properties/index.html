Layer Properties

Just like with individual tile properties, you can define Custom Properties for your tilemap layers. Here's a quick overview of the currently available properties for layers. Be aware these can change at any time, and more will be added!

Property Type Effect
ge_alwaysTop boolean If set to true, layer will always be rendered above characters.
ge_heightShift number This property sets the layer's depth sorting relative to the characters. See the Height Shift guide for more information.
ge_charLayer Beta string This property sets the character layer name for the tilemap layer. See Character Layers for more information.
